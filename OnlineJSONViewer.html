About JSON Validator

JSON Validator is a validator, reformatter and a JSON Viewer for JSON, a lightweight data-interchange format. Copy paste the raw JSON in the editor above and let JSON Validator format, validate and visuallize in tree view, your messy JSON code.

Tips & Tricks

1. You can copy paste the JSON data in editor and click on Format, to format the JSON data.

2. You can click on the "Show Tree View" button to see the data in tree view

3. You can click on "Remove whitespace" button to minifiy the JSON data

Common Errors

1. Expecting 'STRING' - You probably have an extra comma at the end of your collection. Something like { "a": "b",}

2. Expecting 'STRING', 'NUMBER', 'NULL', 'TRUE', 'FALSE', '{', '[' - You probably have an extra comma at the end of your list Something like: ["a", "b", ]

3. Enclosing your collection keys in quotes. Proper format for a collection is { "key": "value" }

4. Make sure you follow JSON's syntax properly. For example, always use double quotes, always quotify your keys, and remove all callback functions.
